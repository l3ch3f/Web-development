1. Bij het inladen van een bestand is het noodzakelijk om de locatie van het
bestand aan te geven. Dit kan op 2 manieren:
● Absolute path
● Relative path
Wat houden deze 2 paths in?
Absolute paths zijn paden waarbij de 'http://www.' worden inbegrepen zoals:
- http://www.mysite.com
- http://www.mysite.com/graphics/image.png
Relative path zijn paden die worden gedefineert vanaf de bron waar het pad begint. Zoals:
- index.html
- /graphics/image.png
- /help/articles/how-do-i-set-up-a-webpage.html


Een image element is voor het inladen van een afbeelding. Afbeeldingen kunnen van
verschillende formaat zijn. Bijvoorbeeld, een afbeelding kan een JPG formaat zijn.
