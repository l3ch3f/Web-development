Opdracht 10

  1. table: Container voor een table element.
  2. thead: Container voor een table head en die komt bovenaan de tabel.
  3. tbody: Container voor de algemene content van de tabel.
  4. tfoot: container voor de voetstuk van de tabel.
  5. th: geeft de table header aan.
  6. tr: rij van de tabel.
  7. td: geeft een data cell aan van het tabel.
