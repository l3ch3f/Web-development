1. charAt()
Zoekt de index van een bepaalde character in een string en returned de index value.
2. concat()
concat() staat gelijk aan de + operator
3. slice()
var res = str.slice(0, 5); laat de index waardes tussen 0 en 5 en de rest wordt vernietigd.
4. split()
var str = "How are you doing today?";
var res = str.split(" ");
verdeeld een string naar een array waarbij elke woord een string zelf is in de array.
5. substr()
var str = "Hello world!";
var res = str.substr(1, 4);
The substr() method extracts parts of a string, beginning at the character at the specified position, and returns the
specified number of characters.
