Opdracht 1 - Basic Syntax

Voordat je Java kan gebruiken moet je de syntax kunnen begrijpen. Doe onderzoek naar de volgende termen en leg uit wat ze (voor Java) betekenen:

  • Object: Een object word gemaakt vanuit een class.
  • Package: Groepeerd classes bij elkaar voor hun relatie met elkaar. Zie het als een folder in een file structuur.
  • Constructor: Een methode(zie het als een functie) om een object te initialiseren. De constructor wordt opgeroepen wanneer een object wordt gecreeerd van een class. Waarbij de attributes worden bepaald.
  • Instance: Zie het als een variabele declaren.
  • Method: Worden binnen een class declared, ze laten bepaalde acties uitvoeren.
  • Class: Een Class is de blauwprint van een object waar het object aan moet voldoen
