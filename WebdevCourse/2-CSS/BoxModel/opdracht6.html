Als je ruimte wilt maken tussen de border en buiten de box, welke css property gebruik je dan? Maak een voorbeeld in je antwoord.

Wij gebruiken hiervoor de margin, omdat dit de ruimte is buiten de border van het geselecteerde element.

De value kan aangegeven worden in verschillende eenheden zoals: rem,px,em en meerdere.
