Welke CSS property kun je gebruiken om ervoor te zorgen dat de waarde van width en height de totale breedte en hoogte is van het blok van de voorgaande opdracht?

Door 100% te gebruiken als value van de box te gebruiken inherit de box de waarde van zijn ouderlijke element. Of simpelweg de value 'inherit' dan is het de default value van de ouderlijke element.
