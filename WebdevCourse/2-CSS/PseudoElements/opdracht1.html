Pseudo Elements

Opdracht 1

Kun je enkele voorbeelden geven van pseudo-elementen (minimaal 4)? Beschrijf met je eigen woorden wat deze pseudo-elementen inhouden.

  • ::after
  • ::backdrop
  • ::before
  • ::selection

::after maakt een pseudo-element dat het kind is van de geselecteerde element. Het is vaak gebruikt om cosmetisch content toe te voegen acteraan een element met de 'content' property. Het heeft inline default settings.

::backdrop backdrop is een box met de grote van de viewport welke gerendered wordt op elke element op de achtergrond daarvan. Vaak goed te gebruiken bij fullscreen mode.

::before Voegt content toe bijvoorbeeld text voor de content van een geselecteerde element.

::selection vergelijkt de proportie van de element dat is geselecteerd door de user.
