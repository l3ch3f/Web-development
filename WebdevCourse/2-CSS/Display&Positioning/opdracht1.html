Display & Positioning

Opdracht 1

Welke position properties zijn er? Leg voor elke position property uit wat het doet. Geef ook aan welke default position waarde elementen hebben die je gebruikt in html.

De 'position' property zijn er als volgt:

  • static
  • relative
  • fixed
  • absolute
  • sticky

Static: Static geeft een positie op de normal flow van de pagina.

Relative: Het element heeft dan een relative positie waardoor het aangepast kan worden.

Fixed: Het element blijft alt op de zelfde plaats zelfs als er gescrolled wordt. En het is ook relatief gepositioneerd.

Absolute: Het element is relatief gepositioneerd aan de dicht bijzijnde ancestor, inplaats van de viewport als fixed.

Sticky: staat gepositioneerd op basis van de users scroll positie.
