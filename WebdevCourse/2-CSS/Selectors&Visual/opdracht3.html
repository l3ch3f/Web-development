Selectors & Visual rules

Opdracht 3

Er zijn verschillende selectors, bijvoorbeeld:

  • type selectors
  • descendant selectors
  • class selectors
Maak van de bovengenoemde selectors voorbeelden. Geef in je antwoord ook aan wat de voordelen zijn van elke selector.

Type Selectors
De [attribute] selector wordt gebruikt met specifieke attribute.

Descendant selectors De descendant selector vergelijkt alle elementen die descendants zijn van een specifieke element.

Class selectors Selecteerd en stijled alle elementen met een class attribute bijvoorbeeld class="intro".
