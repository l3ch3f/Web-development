Opdracht 1

Je kunt op 3 verschillende manieren CSS in je HTML inladen. Op welke 3 manieren kun je dat doen? Leg ook voor elke wijze uit wat het precies inhoud.

  • 1. External CSS: kan je inladen door in de head-section van je meta een link toe te voegen naar een css bestand.
  • 2. Internal CSS: kan je inladen door in de head-section van je meta tussen de style tag toe te voegen.
