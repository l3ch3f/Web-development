Opdracht 8

Leg in eigen woorden uit wat met cascade en inheritance wordt bedoeld. Maak voorbeelden om je antwoord uit te leggen.

  • Cascade

    Cascade is de eerste woord van CSS dat staat voor cascading style sheet.
    Cascade is kort gezegd van boven naar beneden in mijn begrip de volgorde waarbij je je css selector zet van belang is.

  • inheritance

    Inheritance is belangrijk vanwege de erfgoed van een style. Waardoor sommige attributen niet op bepaalde child selectors uitgevoerd zullen worden.
